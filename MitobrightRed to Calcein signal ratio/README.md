# MitoBrightRed to Calcein Signal Ratio

This repository contains a simple image-based analysis workflow to estimate mitochondrial amount per cell by normalizing the MitoBrightRed signal with the Calcein signal.

## Overview

- `C002T001` is treated as the Calcein channel.
- `C003T001` is treated as the MitoBrightRed channel.
- For each sample folder (`*.tif.frames`), the script calculates the mean pixel intensity of both TIFF images.
- The normalized mitochondrial signal is calculated as:

```text
MitoBrightRed / Calcein = mean intensity of C003T001 / mean intensity of C002T001
```

This ratio is used here as an index to estimate mitochondrial amount per cell while correcting for differences in cell-associated signal using Calcein.

## Repository Structure

- `siScr_*_10x.tif.frames`, `siTaz_*_10x.tif.frames`, `siYAP_*_10x.tif.frames`, `siYAPsiTaz_*_10x.tif.frames`
  - Image folders containing the TIFF files for each field/sample.
- `MitobrightRed to Calcein signal ratio/calculate_c003_to_c002_signal_ratios.py`
  - Main analysis script.
- `C003_div_C002_signal_summary.xlsx`
  - Output summary Excel file generated by the script.

## Requirements

- Python 3.10 or later
- `numpy`
- `Pillow`
- `openpyxl`

Install dependencies with:

```bash
pip install numpy Pillow openpyxl
```

## Usage

Run the analysis script:

```bash
python "MitobrightRed to Calcein signal ratio/calculate_c003_to_c002_signal_ratios.py"
```

The script automatically looks for `*.tif.frames` folders either:

- in the same directory as the script, or
- in the parent directory of the script

and writes the result to:

```text
C003_div_C002_signal_summary.xlsx
```

## Output

The generated Excel file contains one row per sample folder and the following columns:

- `folder`
- `c002_file`
- `c003_file`
- `c002_mean_intensity`
- `c003_mean_intensity`
- `c003_div_c002`

## Notes

- The current implementation uses the mean intensity of the entire image.
- No segmentation or per-cell object detection is performed in this script.
- Therefore, the output should be interpreted as a normalized field-level proxy for mitochondrial amount per cell, based on MitoBrightRed corrected by Calcein.
